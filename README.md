# Pulsy Atria Library

The Atria Library is a curated collection of ready-to-use DeFi feed templates for [Pulsy Atria](https://pulsy.app/atria).

This library is constantly evolving as we add new feeds across multiple chains and protocols.

Contributions are welcome! Feel free to create a pull request with new feeds or improvements.

## Feeds

| Feed | Description | Chain(s) |
|------|-------------|----------|
| `eth-transfers-in-range` | Tracks transfers of ETH within a specified amount range. | Ethereum |
| `pol-transfers-in-range` | Tracks transfers of POL within a specified amount range. | Polygon |
| `erc20-transfers-in-range` | Tracks specified ERC20 token transfers within a specified amount range. | Ethereum, Polygon |
